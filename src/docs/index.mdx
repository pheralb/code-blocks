## Prerequisites

Before installing, ensure you have the following:

- [Node.js](https://nodejs.org/en/download/) **v18** or later.
- [React](https://reactjs.org/) **v19** or later.

## Installation

**Codeblocks** requires the following packages:

- Dev dependencies:

```bash
npm install @shikijs/rehype @shikijs/transformers @types/mdx -E -D
```

- Dependencies:

```bash
npm install motion @react-symbols/icons clsx tailwind-merge -E
```

## Utilities

> If you're using [**shadcn/ui**](https://ui.shadcn.com/), you can skip this step.

- Create a custom utility with [clsx](https://github.com/lukeed/clsx) and [tailwind-merge](https://github.com/dcastil/tailwind-merge):

```ts title=src/utils/cn.ts 
import { clsx, type ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

## Styles

- Create a ``codeblock.css`` file with the following content:

```css title=codeblock.css
/* Light & Dark Theme */

html.light code[data-theme*=" "],
html.light code[data-theme*=" "] span,
html.light .shiki,
html.light .shiki span {
  color: var(--shiki-light);
  background-color: transparent !important;
}

html.dark code[data-theme*=" "],
html.dark code[data-theme*=" "] span,
html.dark .shiki,
html.dark .shiki span {
  color: var(--shiki-dark) !important;
  background-color: transparent !important;
}

/* Line Numbers */

code {
  counter-reset: step;
  counter-increment: step 0;
}

code:has(.line:nth-child(2)) .line::before {
  content: counter(step);
  counter-increment: step;
  width: 1rem;
  margin-right: 1.5rem;
  display: inline-block;
  text-align: right;
}

html.light code:has(.line:nth-child(2)) .line::before {
  color: #a0a0a0;
}

html.dark code:has(.line:nth-child(2)) .line::before {
  color: #525252;
}

/* Highlighted Lines */

[data-highlighted-line] {
  position: relative;
  z-index: 0;

  &::before {
    content: "";
    position: absolute;
    top: 0;
    left: -16px;
    right: -16px;
    bottom: 0;
    background-color: #d4d4d4;
    border-left: 1px solid #a3a3a3;
    z-index: -1;
  }

  .dark &::before {
    background-color: #404040;
    border-left: 1px solid #262626;
  }
}
```
