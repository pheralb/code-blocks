The `highlightLines` transformer allows you to highlight specific lines or ranges of lines in your code blocks. This is useful for drawing attention to important parts of your code.

## Usage

Add line numbers or ranges in curly braces `{}` in your code fence meta:

````mdx
```js {1,3-5,7}
const greeting = "Hello"; // Line 1 - highlighted
const name = "World";
function greet() {        // Lines 3-5 - highlighted
  console.log(`${greeting}, ${name}!`);
}
greet();
const farewell = "Goodbye"; // Line 7 - highlighted
```
````

## Syntax

- Single line: `{1}` - Highlights line 1
- Multiple lines: `{1,3,5}` - Highlights lines 1, 3, and 5
- Range: `{1-5}` - Highlights lines 1 through 5
- Combined: `{1,3-5,7-9}` - Highlights line 1, lines 3-5, and lines 7-9

## Examples

### Single Line

````mdx
```ts {2}
function add(a: number, b: number) {
  return a + b; // This line is highlighted
}
```
````

### Multiple Lines

````mdx
```tsx {1,4,7}
import React from 'react';

export function Button() {
  return (
    <button>
      Click me
    </button>
  );
}
```
````

### Line Ranges

````mdx
```js {3-6}
function calculateTotal(items) {
  let total = 0;
  for (const item of items) {
    total += item.price;
  }
  return total;
}
```
````

### Combined Syntax

````mdx
```python {1,3-5,8}
def process_data(data):
    cleaned = []
    for item in data:
        if item is not None:
            cleaned.append(item)
    
    result = analyze(cleaned)
    return result
```
````

## Styling

The transformer adds a `highlighted` class to the specified lines. You can customize the styling in your CSS:

```css
.shiki .line.highlighted {
  background-color: rgba(255, 255, 0, 0.1);
  border-left: 3px solid #fbbf24;
  padding-left: 0.5rem;
  margin-left: -0.5rem;
}
```

## Combining with Other Features

You can combine line highlighting with other transformers:

````mdx
```ts {2-4} title="math.ts" lineNumbers
function multiply(a: number, b: number) {
  // This calculation is highlighted
  const result = a * b;
  return result;
}
```
````

## MDX Integration

To use this transformer in MDX, add it to your Shiki configuration:

```ts
import { highlightLines } from "@/utils/shiki/transformers/highlight-lines";

const rehypeShikiOptions = {
  themes: {
    light: "one-light",
    dark: "one-dark-pro",
  },
  transformers: [
    highlightLines(),
    // ... other transformers
  ],
};
```
