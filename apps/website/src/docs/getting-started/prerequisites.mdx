To create your own custom Code Block component, make sure you have the following prerequisites:

- [Node.js v20](https://nodejs.org/en/download/current) or higher.
- [Typescript v5](https://www.typescriptlang.org/) or higher.
- If you plan to use the [shadcn/ui CLI](https://ui.shadcn.com/docs/cli), make sure you configure the [`components.json`](https://ui.shadcn.com/docs/components-json) file first.

## Styling

All components uses [Tailwind CSS v4](https://tailwindcss.com/docs/installation) for styling. Make sure to have it installed and configured in your project.

For some colors, we use [**neutral** theme](https://tailwindcss.com/docs/colors) from Tailwind CSS:

## Components

1. Create a new React app:

<CreateReactApp />

2. Install the following dependencies:

<CodeBlockSelectPkg
  type="install"
  title="Required Dependencies"
  command="clsx tailwind-merge lucide-react @react-symbols/icons"
/>

3. Add `@` alias to your `tsconfig.json`:

```json {5} title="tsconfig.json"
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

4. Create the `cn` utility:

```ts title="src/utils/cn.ts"
import { clsx, type ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

### Blocks

You'll need to install [@base-ui/react](https://base-ui.com/) to use the Blocks components:

1. Install the package:

<CodeBlockSelectPkg type="install" title="Base UI" command="@base-ui/react" />

2. Add `root` class to global `<body>` tag:

```html title="app/layout.tsx"
<body>
  <div className="root">{children}</div>
</body>
```

3. Add the following CSS to your global stylesheet:

```css title="globals.css"
.root {
  isolation: isolate;
}
```
